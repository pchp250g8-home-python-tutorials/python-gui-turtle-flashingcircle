Программа делает следующее:
  1. Создаёт диалоговое окно (форму).
  2. Загружает исполнителя "Черепашка".
  3. Запускает цикл обработки сообщений (событий) от пользователя и (или) операционной системы.
  4. Исполняются команды исполнителя "Черепашка".
  5. В клиентской области диалогового окна рисует жёлтый эллипс 
  6. При нажатии  кнопки "Закрыть" в заголовке диалогового окна (формы) выгружает исполнителя "Черепашка" и завершает работу.
