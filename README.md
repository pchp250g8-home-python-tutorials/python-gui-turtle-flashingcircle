Программа делает следующее:
  1. Создаёт диалоговое окно (форму).
  2. Загружает исполнителя "Черепашка".
  3. Запускает цикл обработки сообщений (событий) от пользователя и (или) операционной системы.
  4. Выполняются команды исполнителя "Черепашка".
  5. Скрывается курсор исполнителя и отключается его анимация.
  6. Вся клиентская область диалогового окна закрашивается красным цветом.
  7. В центре (позиция "Черепашки" 0, 0) клиентской области диалогового окна рисуется жёлтый круг радиусом 100 пикселей.
  8. Выполняется цикп с параметром.
     8.1. Начальному значению параметра присваивается значение 0, а конечному - 10.
     8.2. Выполняется серия цикла.
          8.2.1. В центре жёлтого круга (позция черепашки 0, 50) рисуется зелёный круг радиусом 50 пикселей.
          8.2.2  Программа делает паузу на 
     8.3. Начальное значение параметра сравнивается с конечным. Если оно меньше, то цикл продолжается. Если нет - завершается.
     8.4. Значение параметра цикла увеличивается на шаг цикла. В данном случае он равен 1.
  10. При нажатии  кнопки "Закрыть" в заголовке диалогового окна (формы) выгружает исполнителя "Черепашка" и завершает работу.
